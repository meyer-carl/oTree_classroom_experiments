# Syllabus Gap Scan

Use this skill when deciding which new experiments to add.

## Workflow

1. Read the current experiment catalog.
2. Map topics to common behavioral and experimental economics syllabi.
3. Identify repeated gaps across sources.
4. Rank additions by teaching payoff and implementation cost.
5. Write the result to `docs/research/syllabus-gap-scan.md` and the backlog files.

## Output

- Source list.
- Topic coverage map.
- Missing topics.
- Prioritized new-app suggestions.
- Notes on anything that is an inference rather than a direct source statement.

## Keep In Mind

- Prefer recurring syllabus topics over niche extras.
- Favor classroom-friendly mechanics.
- Make assumptions explicit when evidence is indirect.
