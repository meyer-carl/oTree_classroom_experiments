Instructions

You will make a series of choices about lotteries (risk), timing (time preferences), and, optionally, loss frames. Each row presents Option A and Option B. Choose the option you prefer in each row.

One row will be randomly selected for payment. If a risk row is selected, the lottery is resolved by a random draw. If a time row is selected, you are paid the amount you chose (the timing label is for classroom discussion). If a loss row is selected, the payoff is based on the sure-loss-versus-gamble choice.
