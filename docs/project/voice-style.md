# Voice And Style

## Tone

- Direct.
- Operational.
- Instructor-facing.
- Low-fluff and high-signal.

## Format

- Prefer short sections.
- Use tables for app catalogs and checklists for launch steps.
- State the consequence of a failure, not just the symptom.

## Content Rules

- Be explicit about headcount, timing, and incomplete-group behavior.
- Name the key fields instructors should inspect in exports.
- Keep debrief guidance tied to the classroom lesson, not the implementation detail.
