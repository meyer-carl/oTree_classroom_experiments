# Project Context Pack

This folder stores the stable memory for the repository.

## Files

- `inventory.md`: what exists today.
- `risk-matrix.md`: where failures are most likely.
- `test-matrix.md`: what should be covered by bots and checks.
- `classroom-failure-modes.md`: live-session recovery patterns.
- `decision-log.md`: choices that should not be re-litigated.
- `voice-style.md`: documentation tone and format.
- `headcount-matrix.tsv`: machine-readable classroom headcount rules for audit and packaging.
- `headcount-flexibility-backlog.tsv`: machine-readable backlog for future uneven-headcount redesigns by app family.
- `instructor-feedback-log.md`: disposition of recurring instructor-facing documentation and release issues.
- `extension-backlog.md`: future app ideas and prioritization.
- `environment.md`: local runtime assumptions and import shadowing notes.
