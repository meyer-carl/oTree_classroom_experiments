# Experiment Catalog

Use this catalog to choose the classroom exercise. Use `01_instructor_pdfs/06_headcount_and_fallbacks.pdf` right after this if your class size may be messy.

## Social Preferences And Bargaining

- `aa_dictator`: baseline dictator game. Legacy preset needs clean pairs.
- `ab_ultimatum`: ultimatum game with direct play and strategy-method fallback. Best current bargaining app when the headcount may be odd.
- `ac_trust`: trust game with strategy-method fallback and configurable multiplier. Also resilient to one spare participant.
- `aj_public_goods`: legacy preset uses trios. The new `public_goods_flexible` preset allows a smaller final group with scaled payoffs.
- `az_endowment_effect`: buyer/seller WTA-WTP comparison. Legacy paired format.
- `ba_gift_exchange`: employer-worker reciprocity exercise. Legacy paired format.
- `ay_volunteer_dilemma`: volunteer dilemma in trios. Legacy exact-multiple format.

## Strategic Interaction

- `ad_nash_demand`: two-player bargaining over a fixed surplus.
- `ae_guess_two_thirds`: beauty contest. The legacy preset runs in pairs; the new `guess_two_thirds_classroom` preset pools the active room into one classroom contest.
- `af_centipede`: centipede game with strategy fallback for odd headcounts.
- `ag_matching_pennies`: two-player mixed-strategy benchmark.
- `ah_coordination`: coordination game in pairs.
- `ai_prisoner_one_rd`: one-shot prisonerâ€™s dilemma in pairs.
- `ai_prisoner_mult_rd`: repeated prisonerâ€™s dilemma with stable pairs across rounds.
- `aw_traveler_dilemma`: travelerâ€™s dilemma in pairs.

## Markets And Industrial Organization

- `ak_market_supply_demand`: eight-trader supply-and-demand market. Treat a missing trader as a hard stop.
- `bb_common_pool_resource`: repeated common-pool extraction game. Legacy groups of 4.
- `bc_asset_market_bubble`: repeated asset market with declining fundamentals. Legacy groups of 4.
- `al_two_sided_auction`: simple buyer-seller trade benchmark. Legacy pairs.
- `am_english_auction`: ascending-price auction. Legacy groups of 4.
- `an_dutch_auction`: descending-price auction. Legacy groups of 4.
- `ao_sealed_bid_first_price`: first-price sealed-bid auction. Legacy groups of 4.
- `ap_sealed_bid_second_price`: second-price sealed-bid auction. Legacy groups of 4.
- `aq_ebay_auction`: eBay-style proxy bidding. Legacy groups of 4.
- `av_common_value_auction`: common-value auction. Legacy groups of 4.
- `at_bertrand`: Bertrand competition in pairs.
- `au_cournot`: Cournot competition in pairs.

## Individual Measurement

- `ar_risk_time_preferences`: risk, time, loss, and ambiguity preferences. Best fallback when a multiplayer activity cannot form cleanly.
- `as_competitiveness`: timed competitiveness task. Legacy groups of 4.

## Support Apps

- `survey`: safe follow-up or fallback app for any headcount.
- `payment_info`: redemption and payout instructions for any headcount.
